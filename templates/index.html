Редактор Python-кода

        
      

      

        

          
Как работать

          
    
            
  • Редактор — открыть текстовый режим и править код вручную.
    • 
            
  • Собрать — превратить код в блоки на рабочей области.
    • 
            
  • Закрыть — выйти из редактора без сборки блоков.
    • 
            
  • Код из блоков и из редактора — один и тот же: после сборки блоки заменяются новой программой.
    • 
          

        

        

          
Удобства редактора

          
    
            
  • Подсветка ключевых слов, строк, чисел и комментариев.
    • 
            
  • Tab — отступ (4 пробела); Shift+Tab — убрать отступ.
    • 
            
  • Enter после строки с : — автоматический отступ внутри блока.
    • 
            
  • Backspace в начале строки — снять отступ целиком.
    • 
          

        

        

          
Что можно писать

          
    
            
  • Переменные, print, input, арифметика и сравнения.
    • 
            
  • if / elif / else, циклы for и while.
    • 
            
  • break, continue, списки, строки и их методы.
    • 
            
  • import math, import random, функции из этих модулей.
    • 
          

        

        

          
Ограничения

          
    
            
  • Поддерживается только то, для чего есть блоки в конструкторе.
    • 
            
  • Нельзя: свои функции (def), классы, from … import, сложные выражения в одной строке.
    • 
            
  • Если «Собрать» выдаёт ошибку — упростите код или соберите программу из блоков.
